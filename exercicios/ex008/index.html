Principais formatações

Negrito / Destaque

Nesta frase temos um negrito usando a tag "b" (não semântica).

Nesta frase temos um termo em destaque usando a tag "strong" (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag "i" (não semântica).

Nesta frase temos um termo em ênfase usando a tag "em" (semântica).

Texto marcado

Podemos criar, também, um texto marcado, usando a tag "mark".

Podemos alterar a cor do texto marcado, usando estilos.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo, usando as tags "big" (tag obsoleta) e "small".

Texto deletado

Podemos marcar um texto como exluído, para indicar que ele deve ser lido, mas não considerado.

Texto inserido / Sublinhado

Podemos marcar um texto como inserido, para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Existe, também, o texto sublinhado, usando a tag "u" (não semântica).

Texto sobrescrito

Para inserir algo do tipo x2+3x+5.

Texto subscrito

Para inserir algo do tipo H2O.
